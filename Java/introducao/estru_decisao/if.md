#### If;  else if;  else

```java
if(3==1){
	System.out.println("teste");
}
else if(3==1){
	System.out.println("teste2");
}
else{
	System.out.println("teste3");
}

//Ou para estrutura mais simples quando se tem apenas 1 comando após as condições:

if(3==1) System.out.println("teste1");
else if(4==4) System.out.println("teste2");
else System.out.println("teste3");
```
