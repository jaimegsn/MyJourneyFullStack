#### Operadores de Atribuição

Usado para atribuir valores em uma variável

- '='   Equivalente a  x = 1
- '+='  Equivalente a  x = x + number
- '-='  Equivalente a  x = x - number
- '*='  Equivalente a  x = x * number
- '/='  Equivalente a  x = x / number
- '%='  Equivalente a  x = x % number
- '++' Incrementador Equivalente a x=x+1;
- '--' Decrementador Equivalente a x=x-1;

Exemplos:

```java
var x = 5;
x+=2;
x/=3;
..
```
